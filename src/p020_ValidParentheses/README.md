# p020_Valid Parentheses

###### tags: `LeetCode` `Easy`

## 題目
[原題目連結](https://leetcode.com/problems/valid-parentheses/)

給定一個由 `() {} []` 所組成的字串s，請你求s是否合法字串。

* 合法字串：`{}()[]`、`([{}])`
* 不合法字串：`][`、`())`、`({)}`、`(](}`
